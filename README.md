### Instalación


1. Clona el repositorio

   ```sh
   git clone https://github.com/JaviiAzagra/javierazagra.com-astro.git
   ```

2. Instala los paquetes de NPM

   ```sh
   pnpm install
   ```

3. Ejecuta el proyecto

   ```sh
   pnpm run dev
   ```
